Cheat Sheet

  • Close Tabs to the Right: "alt + w"
  • Duplicate Tab: "alt + d"
  • Highlight Tabs Left: "alt + shift + left"
  • Highlight Tabs Right: "alt + shift + right"
  • Move Common Domain Tabs: "alt + shift + m"
  • Move Tab Left: "alt + shift + ["
  • Move Tab Right: "alt + shift + ]"
  • Move Tab To Position: "alt + {1-9}"
  • Move to New Window: "alt + m"
  • Pin Tab: "alt + p"
  • Refresh All Tabs: "alt + r"
